# 本地摄像头监控系统 - 需求文档

## 1. 项目概述

构建一个基于本地USB摄像头的视频监控与存储系统。系统通过FFmpeg将摄像头采集的视频流模拟为RTSP流，利用Netty进行RTSP流的接收与分发，Spring Boot作为后端服务框架，最终将视频流存储到MinIO对象存储中，并提供视频回放与检索功能。

## 2. 业务目标

- 实现本地USB摄像头的实时视频监控
- 将视频流持久化存储到MinIO
- 支持按时间、摄像头等维度对历史视频进行检索和回放
- 提供可扩展的架构，未来可支持多摄像头接入

## 3. 功能需求

### 3.1 视频采集

| 编号 | 需求项 | 说明 |
|------|--------|------|
| F-01 | 摄像头接入 | 支持USB摄像头接入（Linux系统使用V4L2，Windows使用DirectShow） |
| F-02 | RTSP推流 | 使用FFmpeg将摄像头原始视频流封装为RTSP协议推送至服务端 |
| F-03 | 流参数配置 | 支持配置视频分辨率、帧率、编码格式（H.264）、码率等参数 |
| F-04 | 多路扩展 | 架构上支持多路摄像头同时推流（当前实现单路，预留多路能力） |

### 3.2 视频流服务

| 编号 | 需求项 | 说明 |
|------|--------|------|
| F-05 | RTSP接收 | 基于Netty实现RTSP Server，接收来自FFmpeg的推流 |
| F-06 | 流媒体处理 | 解析RTSP/RTP协议，提取H.264视频帧数据 |
| F-07 | 实时预览 | 支持将实时流通过WebSocket或HTTP-FLV分发给前端进行预览 |
| F-08 | 连接管理 | 管理RTSP会话生命周期（DESCRIBE、SETUP、PLAY、TEARDOWN） |

### 3.3 视频存储

| 编号 | 需求项 | 说明 |
|------|--------|------|
| F-09 | 分片存储 | 将视频流按固定时长（如10分钟）切割为TS分片文件 |
| F-10 | MinIO上传 | 将视频分片上传至MinIO对象存储 |
| F-11 | 索引记录 | 在数据库中记录每个分片的元数据（时间范围、摄像头ID、文件路径、时长等） |
| F-12 | 存储策略 | 支持配置存储保留天数，过期分片自动清理 |

### 3.4 视频回放与检索

| 编号 | 需求项 | 说明 |
|------|--------|------|
| F-13 | 时间范围检索 | 支持按起止时间查询历史视频片段 |
| F-14 | 摄像头筛选 | 支持按摄像头ID筛选视频 |
| F-15 | 视频回放 | 支持按时间线回放历史视频（通过合并分片或生成播放列表） |
| F-16 | 播放控制 | 支持暂停、快进、倍速播放等控制 |

### 3.5 系统管理

| 编号 | 需求项 | 说明 |
|------|--------|------|
| F-17 | 状态监控 | 显示摄像头在线/离线状态、推流状态、存储空间使用情况 |
| F-18 | 日志记录 | 记录系统运行日志、异常日志、操作日志 |
| F-19 | 配置管理 | 支持通过配置文件管理RTSP端口、MinIO连接、FFmpeg参数等 |

## 4. 非功能需求

| 编号 | 需求项 | 说明 |
|------|--------|------|
| NF-01 | 性能 | 支持单路1080P@25fps实时流的接收、处理与存储 |
| NF-02 | 延迟 | 实时预览端到端延迟不超过3秒 |
| NF-03 | 可靠性 | 断流后支持自动重连，异常恢复后继续录制 |
| NF-04 | 存储效率 | 1080P视频每小时存储不超过2GB（合理码率控制） |
| NF-05 | 兼容性 | 支持Linux和Windows部署环境 |
| NF-06 | 安全性 | MinIO访问需认证，RTSP服务可选配置认证 |

## 5. 技术栈

| 组件 | 技术选型 | 用途 |
|------|----------|------|
| 后端框架 | Spring Boot 3.x | Web服务、业务逻辑、REST API |
| 网络通信 | Netty 4.x | RTSP Server实现、流媒体数据传输 |
| 视频处理 | FFmpeg | 摄像头采集、编码、RTSP推流 |
| 对象存储 | MinIO | 视频分片文件存储 |
| 数据库 | MySQL / PostgreSQL | 视频索引元数据存储 |
| 实时通信 | WebSocket | 前端实时预览 |
| 构建工具 | Maven | 项目构建与依赖管理 |

## 6. 术语表

| 术语 | 说明 |
|------|------|
| RTSP | Real Time Streaming Protocol，实时流传输协议 |
| RTP | Real-time Transport Protocol，实时传输协议 |
| TS | MPEG-TS，MPEG传输流格式 |
| H.264 | 视频编码标准 |
| MinIO | 高性能分布式对象存储，兼容S3 API |
| V4L2 | Video4Linux2，Linux下的视频采集框架 |
| DirectShow | Windows下的多媒体流处理框架 |
